# SEO Improvement Log Template

Use this when the project has no stronger existing format. Keep entries concise but auditable.

```markdown
## YYYY-MM-DD - Short action title

**Site**
- Domain:
- Publishing mode:
- Business goal:

**Baseline**
- Source: GSC / RefreshAgent / Bing / analytics / keyword tool / other
- Date range:
- URL or query:
- Clicks:
- Impressions:
- CTR:
- Average position:
- Cache age or data freshness:

**Authority Mark** (if used)
- Target query:
- Site/domain DR:
- First lower-DR top-10 domain:
- Rank:
- DR:
- Unavailable DR values:
- Decision impact:

**Decision**
- Chosen action:
- Why this action beat alternatives:
- Existing URL optimized or new URL created:

**Implementation**
- Publishing path:
- Records/files changed:
- Touched URLs:
- Sources used:

**Verification**
- Rendered page status:
- Indexability/canonical:
- Title/meta:
- Schema:
- Internal links:
- Tests/checks:

**Indexing**
- Method:
- URLs submitted or exposed:
- Result:

**Follow-Up**
- Recheck date:
- Metric to compare:
- Open risks or next ideas:
```

## Minimum Entry

If time is short, record at least:

- date
- target URL/query
- baseline clicks, impressions, CTR, position, and date range when available
- Authority Mark when used
- action taken
- touched URL
- verification result
- indexing result
- follow-up date
